CSS, or Cascading Style Sheets, is a powerful styling language that can be used to create visually stunning designs and effects on websites and applications. One of the features of CSS that I’ve recently been exploring is the use of em units.

For those who may not be familiar, em units are a way to define the size of text or other elements in relation to the size of the font of the parent element. This allows for greater flexibility in designing responsive layouts, as the size of elements will automatically adjust based on the size of the font.

Recently, I’ve been experimenting with using em units in my Waybar config, a status bar for Wayland compositors. By using em units, I can create designs that automatically adjust to the size of the font, making it easier to create visually pleasing layouts that look great on different screen sizes and resolutions.

But what really caught my attention was the ability to use custom keyframes to create gradient animations in my waybar. This is a really cool feature that allows me to create animations that change the color of my status bar over time. It’s a really simple yet effective way to add a touch of personality and style to my setup.
