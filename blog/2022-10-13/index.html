I’ve been diving deeper into the world of Hugo lately, and it’s been a really cool experience. The ability to quickly create a static website using markdown and pre-designed themes makes it a breeze to put together a site without too much hassle.

However, I’ve run into a few issues along the way. For example, getting a Freenom domain has been a bit more complicated than I expected. I’ve also had some trouble with the latest update to Waybar, which ended up messing with my style.css and causing a few headaches. But, I was able to fix it by downgrading some dependencies and using the Waybar-Hyprland-git package.

On the bright side, I’ve been using Hyprland as my daily driver for the past two months and it’s been a really pleasant experience. It’s not without its bugs, but I’m confident that they’ll be ironed out in the future. This is how my Desktop looks now –>
